- MLOps는 Machine Learning + Ops로 머신러닝 모델을 실제 환경에 적용하며 겪는 다양한 운영 이슈와 관련된 내용
- 정의는 사람마다 다르고, 어떤 분들은 AIOps로 부르기도 함
- 자세한 내용은 [MLOps Wikipedia](https://en.wikipedia.org/wiki/MLOps) 참고
